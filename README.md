# 🚀 AutoSRE AI – Multi-Agent Incident Resolver

## 📌 Overview
AutoSRE AI is a multi-agent system that automates incident detection and resolution using AI.

## 🧠 Features
- Log Analysis using AI
- RAG-based known issue detection
- GitHub MCP integration for new issues
- Automated task creation (Jira/Slack simulated)
- Multi-agent orchestration

## 🏗️ Architecture
- Orchestrator Agent
- Log Analysis Agent
- RAG Agent
- GitHub MCP Agent
- Action Agent
- Task Agent

## ⚙️ Tech Stack
- FastAPI
- GCP Cloud Run
- FAISS (RAG)
- GitHub API
